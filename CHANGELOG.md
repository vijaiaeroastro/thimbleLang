# Changelog

## 0.1.0

Initial development release.

- Header-only C++17 interpreter
- Scalar values, arrays and maps
- Lexical bindings, functions and control flow
- Typed C++ function and member bindings
- Opaque host objects with registered properties, direct fields and methods
- Typed `Result<T>` host returns and compile to execute interface validation
- Structured diagnostics and execution limits
- Named source diagnostic rendering and location-aware call traces
- Step, call-depth, collection, string, allocation, time and cancellation limits
- Single-header amalgamation
- Geometry and application-policy examples
- Cycle-safe equality and rejection of cycle-forming script mutations
- Exact expression spans and uniform host-boundary error conversion
- Optional Clang coverage-guided fuzz target
- CMake consumer target, sanitizer checks and runtime benchmark
- Ubuntu, Windows and macOS continuous integration
